What does ChemAPI do?

ChemAPI is a free API that allows developers to search for the:

  • boiling point and melting point
  • density
  • molecular weight/formula
  • and more

of tens of thousands of common laboratory chemicals.

We built this API to give back to the scientific community, because PhDs are awesome.
This project is funded out of our paychecks, so if you like what we're doing, please consider donating :).

What can I use ChemAPI for?

Example Uses:

  • Source of chemical data for electronic lab notebooks.
  • Automatically gather chemical data for chemical inventories.
  • Investigate relationships between chemical properties and molecular structure using ChemAPI as a data source
  • Program chemical knowledge into voice assistants like Google Assistant, Alexa, or Mycroft.
  • Basically, add chemical property data to whatever your heart desires.

Does ChemAPI cost money?

ChemAPI is free to use!

Our philosphy is that scientific data needs to be as easily accessible as possible, so ChemAPI is and always will be free for the scientific community for eternity/until the end of the universe.

Docs

The /lookup endpoint returns the chemical properties, in the json format, of a chemical that you input.

You can either input the chemical as a CAS number or a chemical name.

We can even handle misspellings, chemical "vernacular" (for example baking soda), and some trade names!

There are many ways to query for a chemical:

https://54v06pqby9.execute-api.us-east-1.amazonaws.com/dev...

/lookup?q=caffeine (correct spelling)
/lookup?q=cafiene  (misspelling)
/lookup?q=No Doz  (brand name)
/lookup?q=58-08-2  (CAS number)
/lookup?q=C8H10N4O2  (molecular formula)
/lookup?q=1,3,7-Trimethylpurine-2,6-dione  (IUPAC)

Returns all available chemical properties for caffeine in the json format (see below).

There are many ways to search for a single compound. Also, as an added convenience for developers and their end users, our API is able to effectively handle typos!

{

   "boiling_point_C":178.0,
   "density_gcm-1":1.23,
   "melting_point_C":236.5,
   "mw":194.19,
   "hazards":[  
      "Harmful",
      "Highly Flammable",
      "Toxic"
   ],
   "prod_name":"Caffeine",
   "mol_formula":"C8H10N4O2",
   "physical_description":"white to light yellow crystal powder",
   "color":"Silky white or white"
}

Details about the /lookup endpoint:

This endpoint searches for and returns properties for the chemical that most closely matches your query in our dataset. Think of it like an "I'm feeling lucky" search for chemicals.

key description of the key
boiling_point_C boiling point returned as a float with units of degrees Celcius
melting_point_C melting point returned as a float with units of degrees Celcius
density_gcm-1 density returned as a float with units of g/mL
density_is_estimated returns true if the density is not determined experimentally (otherwise does not appear)
mw molecular weight returned as a float with units of g/mol
prod_name the chemical name returned as a string
mol_formula the molecular formula returned as a string
physical_description the physical description (apppearance) at STP
color the color at STP
hazards hazards returned as a list of strings

Usage Limits

Feel free to use this service without an API key for up to 10,000 requests per day. However, if use exceeds 10,000 requests per day, keyless requests may be rate-limited without notice. If you plan on using more than 10,000 requests per day, you can get an API key and use get more stable access to our API. API keys are free for noncommercial use. For API key access, send us an email and tell us about your application and include a traffic estimate.

If you use our free API, please give us a shoutout. Thank you!!!

Future Work

We're attempting to build some new client libraries and add this to the major API marketplaces so that more people can get easy access to this API.

Also, we're looking into creating a few new endpoints like /search, /suggest, and (what we're most excited about) /predict, which will be powered by machine learning. Check back every so often for progress reports!
